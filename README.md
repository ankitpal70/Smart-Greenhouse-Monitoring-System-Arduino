# Smart Greenhouse Monitoring System

## Overview

The Smart Greenhouse Monitoring System is an Arduino-based project designed to monitor environmental conditions inside a greenhouse. It uses a DHT11 sensor to measure temperature and humidity in real time and helps maintain suitable conditions for plant growth.

## Features

* Real-time temperature monitoring
* Real-time humidity monitoring
* LCD display output
* Automatic alert system using buzzer
* Cost-effective and easy to implement

## Components Used

* Arduino Uno
* DHT11 Temperature & Humidity Sensor
* 16x2 LCD (I2C)
* Buzzer
* Jumper Wires
* Breadboard

## Working Principle

The DHT11 sensor continuously reads temperature and humidity values. The Arduino processes the data and displays it on the LCD screen. If the temperature exceeds a predefined threshold, the buzzer activates to alert the user.
